## 题目 4：图书借阅信息管理系统

图书馆是大学学习的重要资源，充分利用馆藏是每位大学生的基本技能。  
系统需模拟并管理图书借阅信息，支持 ≤1000 条记录，单册复本不限，以菜单驱动完成以下功能。

---

## 功能需求

### 1. 图书借阅信息录入
- 模拟学生借书流程，录入格式示例：

| 字段       | 示例值         |
|------------|----------------|
| 图书名称   | 机器学习在线   |
| 价格       | 76.0           |
| 借阅日期   | 2025-10-12     |
| 流水号     | 20251012112658 |
| 借阅人     | 张明媚         |
| 借阅人ID号 | 2025001        |

- 录入一条后提示“是否继续录入(Y/N)”，支持批量输入；结束后写入文件持久化。

### 2. 图书借阅信息排序输出
- 读取文件全部记录，按**借阅人ID号降序**排列并打印。  
- 每条记录之间用分割线分隔，清晰直观。

### 3. 按图书名称查询借阅记录
- 输入书名 → 输出该书全部借阅记录（允许多条）。  
- 未找到时给出提示；记录间用分割线分隔。

### 4. 按借阅人姓名查询借阅记录
- 输入姓名 → 输出该人全部借阅记录。  
- 未找到时给出提示；记录间用分割线分隔。

### 5. 查询价格最高和最低的图书
- 遍历所有借阅记录，找出价格最高与最低的图书（假设无同价冲突）。  
- 分别输出其完整信息。

### 6. 统计输出借阅次数最多的 3 本书
- 对图书出现次数进行统计，输出**借阅次数 TOP3** 的书名及对应次数。
