# BookBorrowManageSystem
大一上期末作业设计

基于C语言实现的图书借阅信息管理系统，支持借阅信息录入、排序、查询、统计等核心功能
 
功能介绍
 
1. 图书借阅信息录入：支持批量录入借阅记录，录入完成后持久化到文件

2. 借阅信息排序输出：按借阅人ID号从大到小排序并展示所有记录

3. 按图书名称查询：输入书名，查询该图书的所有借阅记录

4. 按借阅人姓名查询：输入姓名，查询该用户的所有借阅记录

5. 价格最值查询：统计并展示价格最高和最低的图书信息

6. 借阅次数统计：分析借阅记录，输出借阅次数排名前三的图书

题目4 图书借阅信息管理系统
图书馆是大学学习的一个重要资源，学会并充分利用图书馆的学术资源是每一个大学生必须掌握的基本技能，我国很多高校都有丰富的藏书，藏书量的多少往往是一所大学地位和名望的标志之一，图书馆对学生的学习研究极有帮助，极为重要，现要求模拟并设计一个图书馆的图书借阅信息管理系统，设定图书的借阅信息记录不超过1000，每本图书可供借阅的数量不限，要求以菜单选项的方式完成相应的功能，主要功能需求如下：

(1)图书借阅信息录入

要求录入图书的借阅信息模拟学生图书借阅的过程，图书借阅信息是信息管理的基础数据，也是数据分析统计的重要来源，图书借阅信息录入参考如下格式：

图书名称：机器学习在线

价格：76.0

借阅日期：2025-10-12

流水号:20251012112658

借阅人：张明媚

借阅人ID号：2025001

录入一条借阅记录后，程序应提示是否录入下一项(Y/N)，以便用户可以录入多条记录，信息录入完毕后写入文件。

(2)图书借阅信息排序输出

根据读取文件图书借阅记录，按借阅人ID号从大到小的顺序排序输出所有的借阅记录，各借阅记录用分割线作为分隔符，使信息展示清晰直观。

(3)按图书名称查询借阅记录

输入图书名称查询所有的借阅记录，如果有多个借阅记录(同一本书可能有多个借阅记录)，应全部输出显示，记录之间用分割线作为分隔符，如没有找到借阅信息应给出提示信息。

(4)按借阅人姓名查询借阅记录

输入借阅人姓名查询所有的借阅记录，如果有多个借阅记录，应全部输出显示，记录之间用分割线作为分隔符，如没有找到借阅信息应给出提示信息。

(5)查询价格最高和最低的图书

根据图书借阅信息查询价格最高的图书和价格最低的图书，假设不存在价格相同的图书。

(6)统计输出借阅次数最多的3本书

对借阅记录进行统计分析，同一种图书可能有多条借阅记录，找出借阅次数排前3名的图书，并输出显示它们的借阅次数。
